# llm-json-repair

> **LLM Output JSON Repair Tool:** A zero-dependency utility to sanitize and repair invalid, commented, trailing-comma, markdown-wrapped, or truncated JSON strings generated by LLMs.

---

## ⚡ Features
- **Zero Runtime Dependencies**: Ultra lightweight and fast character-scanning.
- **Markdown Fences Stripper**: Automatically detects and extracts JSON from ` ```json ... ``` ` blocks.
- **Comments and Quotes Cleaner**: Strips single-line (`//`) and multi-line (`/* */`) comments. Standardizes single quotes to valid JSON double quotes.
- **Trailing Commas Remover**: Safely cleans trailing commas in lists and objects.
- **Truncated JSON Recovery**: Automatically balances unclosed braces, brackets, and quotes when the LLM stream runs out of tokens.

---

## 📦 Installation

```bash
npm install llm-json-repair
```

---

## 🚀 Usage

### 1. CLI Usage
Print the repaired JSON directly to standard output:

```bash
npx llm-json-repair '{"status": "ok", "users": ["John'
# Output: {"status": "ok", "users": ["John"]}
```

### 2. Programmatic API

```javascript
import { parseJson } from 'llm-json-repair';

const dirtyLlmResponse = `
  \`\`\`json
  {
    // config settings
    "name": 'app',
    "features": ['auth', 'billing', ], // trailing comma
  }
  \`\`\`
`;

const parsed = parseJson(dirtyLlmResponse);
console.log(parsed.name); // "app"
console.log(parsed.features); // ["auth", "billing"]
```
